# Plus — браузер для Windows (MVP)

**Plus** — нативный браузер для Windows 10/11 с интерфейсом на egui и контентом в WebView2. Адблок и VPN работают через локальный HTTP‑прокси, который может цепляться к SOCKS5 sing-box.

## Возможности
- Нативный интерфейс: вкладки, навигация, адресная строка, диагностика
- WebView2 только для контента страниц
- AdBlock (ABP‑совместимые правила) + статистика блокировок
- VPN через sing-box (локальный SOCKS5) + проверка IP
- История и закладки (минимум)
- Загрузки с предупреждением для опасных файлов

## Быстрый старт (Windows)

## Проверка функций (кратко)
1. Запуск: `cargo run -p plus-desktop`
2. Откройте 2 вкладки, перейдите на yandex.ru
3. Откройте «Диагностика» и убедитесь, что счётчик AdBlock растёт
4. Если настроен VPN — нажмите «Check IP»

## Архитектура
Поток: **WebView2 → локальный AdBlock‑proxy → (опционально) VPN SOCKS5 → Интернет**.
